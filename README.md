## Core Architecture
A high-performance spatial interface operating entirely within the browser. It maps raw 21-point neural hand-tracking data directly to a 4,800-particle buffer geometry matrix with zero latency. When user input ceases, the engine smoothly transitions into a procedural state machine, executing autonomous trigonometric flow fields and kinetic repulsions.

## HCI Gesture Mapping

| Input | Output |
| :--- | :--- |
| **1-5 Fingers** | Topology Morph (Heart, Saturn, Flower, Mirror, Fireworks) |
| **Thumb/Index Pinch** | Matrix Scaling (Zoom) |
| **One Pinch + One Palm** | Manual Rotation Override |
| **Two Open Palms** | Continuous Hue Oscillation |
| **Fist Contraction** | Gravitational Pull (Compress) |
| **Rapid Expansion** | Kinetic Velocity (Blast) |

## Autonomous Anomaly Engine
Without user input, the system continuously generates fluid dynamics and complex mathematical structures:
* **Swarm:** Fluid trigonometric flow fields.
* **Entity Clash:** Binary particle separation and collision.
* **Geometry:** Procedural Torus Knot generation.
* **Signal:** Particle-rendered cryptographic text.
